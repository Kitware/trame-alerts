# trame-alerts

A trame package to manage and display alerts

* Free software: Apache Software License

## Installing
Install `trame_alerts` using `pip`
```bash
pip install trame_alerts
```

## Developing

Build and install the core Vue components

```bash
    cd vue-components/core
    npm i
    npm run build
    cd -
```

Install the library

```bash
pip install -e ".[dev]"
```
